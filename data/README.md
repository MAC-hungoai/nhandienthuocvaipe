# Dữ Liệu Cục Bộ

Thư mục `data/` dành cho dữ liệu không nên đẩy thẳng lên Git, ví dụ:

- ảnh thật bổ sung của người dùng
- bootstrap output
- file trung gian khi fine-tune

## Cấu trúc thường dùng

```text
data/
└── user_real_photos/
    └── pill/
        ├── image/
        └── label/
```
